## 架构变更：单循环 → 双循环

```
while(g_running){                        // 外层：运动检测
    frame = ring_buffer_get(rb)
    YUYV→RGB + shm 写帧
    if(motion_detect_process(...)){
        SMS + MQTT 各发一次（上升沿）
        frame_free(frame)
        while(g_running){                // 内层：报警（不检测运动）
            frame = ring_buffer_get(rb)
            YUYV→RGB + shm 写帧
            LED 亮（状态变化才写 sysfs）
            warning=1
            JPEG 存图
            if(cancel 来了){
                LED 灭, warning=0
                frame_free(frame)
                break                  // 退出内层
            }
            frame_free(frame)
        }
        continue                        // 回到外层
    }
    frame_free(frame)
}
```

## 遗留
- cancel 后连续运动会重触发（外层下一帧又检测到运动）
- 帧率 ~3fps（Cortex-A7 YUYV→RGB + 900KB shm memcpy 物理瓶颈）
- GUI 左右打架（用户暂时忽略）
- 帧率状态栏已删除
