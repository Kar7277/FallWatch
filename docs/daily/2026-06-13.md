## 今天踩过的坑 & 修复记录

### ❌ 错误 1：goto 跳过 struct 初始化
- **问题**：ring_buffer_init 的 goto 在 struct 声明前面，跳过所有 ={0} 初始化
- **根因**：C 语言 goto 跳过初始化声明时变量值不确定
- **修复**：所有 struct 声明移到第一个 goto 前面

### ❌ 错误 2：sms 声明被 goto 跳过
- **问题**：sms 声明在 ring_buffer_init 的 goto 之后，cleanup 中垃圾值
- **修复**：sms_alert_init 调用提前到所有 goto 之前

### ❌ 错误 3：SMS/MQTT 失败也打印 SUCCESS
- **问题**：软依赖没有 goto，代码落到无条件 SUCCESS 行
- **修复**：用 else 分支包裹 SUCCESS 打印和 pass++

### ❌ 错误 4：Qt Creator StatusDot 占用空间问题
- **问题**：ui 布局中 StatusDot 自定义 widget 无空间占用，拖拽失败
- **根因**：setFixedSize(24,24) 只是控件，但 Designer 不识别
- **处理**：先用 QLabel 占位，后代码替换为 StatusDot

### ❌ 错误 5：jpeg_save_frame out_path 覆盖 js.output_dir
- **问题**：每次保存后目录路径被文件名覆盖
- **根因**：out_path 和 js->output_dir 指向同一块内存
- **修复**：用独立局部 buffer 承接输出路径

### ❌ 错误 6：ring_buffer_init 被调用两次
- **问题**：`rb = ring_buffer_init(8); g_rb = rb = ring_buffer_init(8);`
- **根因**：编辑失误，第一行没删
- **修复**：删掉第一行，只保留 `g_rb = rb = ring_buffer_init(8);`

## 今天学到的新知识

### ⭐ 1. QT 信号槽连接 (Qt5)
- `SIGNAL(timeout())` / `SLOT(onBlinkTimer())` 宏写法兼容性最好
- Lambda connect 在 Qt5 某些版本有重载歧义
- 需要 `private slots:` 独立声明区域

### ⭐ 2. QPainter 自定义 widget 绘制
- `paintEvent` 重写 + `QPainter` 画形状
- `setRenderHint(Antialiasing)` 抗锯齿
- `update()` 触发重绘

### ⭐ 3. Qt Designer 布局拖拽
- 右键 Lay out 对空容器无效
- 先拖 Layout widget 到容器，再往里面放控件

### ⭐ 4. MQTT + ADB 端口转发
- `adb forward tcp:1883 tcp:1883` 实现 USB 线传 MQTT
- 板子 localhost broker → ADB → PC localhost 订阅

### ⭐ 5. BusyBox date 时间设置
- `date -s "YYYY-MM-DD HH:MM:SS"` 格式化设置
- 不认 Unix 时间戳参数
